Welcome to CartesIO’s documentation!¶

The CartesIO library (a.k.a. CartesianInterface, which was its codename during development), is an awesome library for robotic applications.

Features¶

  • a flexible hardware abstraction layer, with multi-framework support (notably, ROS)
  • a real-time plugin handler to develop custom RT controllers
  • useful abstraction for inter-plugin communication
